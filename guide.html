시작하기

회원가입부터 로그인까지

회원가입
1로그인 화면 하단 회원가입 클릭
2이메일, 이름, 직급, 비밀번호 입력
3보안 질문 & 답변 설정 (비밀번호 분실 시 사용)
4가입하기 클릭 → 로그인 화면으로 이동
로그인
1이메일, 비밀번호 입력
2로그인 정보 저장 체크 시 다음부터 자동 입력
3로그인 클릭
메인 메뉴
로그인 후 2×3 그리드 메뉴가 표시됩니다
📦 재고 관리, 🚗 출장/차량, 📢 공지사항, 💳 법인카드 중 원하는 모듈을 선택
"준비중" 칸은 향후 추가될 기능입니다
💡 TIP

비밀번호를 잊었다면 비밀번호를 잊으셨나요?를 눌러 보안 질문으로 초기화하세요.

⚠️ 중복 로그인 차단

다른 기기에서 같은 계정으로 로그인하면 기존 기기는 자동 로그아웃됩니다. 한 번에 하나의 기기에서만 사용 가능합니다.

📦

재고 관리

부품 등록, 입출고, 이력 조회

부품 등록
1재고 목록에서 + 버튼(파란 동그라미) 클릭
2부품 유형(상용품/가공품) 선택
3종류, 품목명*, 규격, 도면번호, 메이커*, 공급처*, 단가 입력
4재고 수량, 안전재고, 보관위치, 사용처 입력
5사진·도면 첨부 (카메라 촬영 또는 갤러리 선택)
6저장 클릭 → 초기 수량이 있으면 입고 이력이 자동 생성됩니다
입출고 처리
1재고 목록에서 부품 카드 클릭 → 부품 상세 페이지
2하단 + 입고 또는 − 출고 클릭
3수량, 사유 입력 후 확인
재고 검색 & 정렬
상단 드롭다운 필터 → 정렬(종류순·이름순·수량순·최신순), 카테고리 선택
검색창에서 품목명, 메이커로 검색
이력 탭
상단 📋 이력 탭 → 입출고 내역 조회
입출고 구분 / 기간 필터로 원하는 이력만 확인
CSV 내보내기 버튼으로 엑셀 다운로드 가능
💡 TIP

안전재고 이하로 떨어지면 수량이 빨간색으로 표시됩니다.

출장 / 차량

출장명령서, 차량 관리, 소모품

출장명령서 작성
1출장 탭 → 우측 하단 + 버튼 클릭
2목적지, 출장목적, 출발·복귀 일시 입력
3교통수단 선택 (회사 차량 / 개인 차량 / 대중교통)
4회사 차량 선택 시 → 차량 목록에서 선택 (자동 예약)
5동행자 추가 (선택)
6저장 클릭 → 상태: 예약됨
출장 상태 흐름
작성 → 예약됨 → 열쇠 수령 시 진행중 → 열쇠 반납 시 완료
차량 현황 (차량 탭)
차량별 상태(대기/예약됨/사용중/정비중) 확인
열쇠 수령: 차량 카드의 수령 버튼 클릭 → 주행거리 입력
열쇠 반납: 차량 카드의 반납 버튼 클릭 → 주행거리·주유잔량 입력
차량 상세
차량 카드 클릭 → 상세 정보, 엔진오일 교체 기록, 주유 이력, 월별 주행거리 확인
차량 소모품 관리
소모품 탭에서 에어필터, 워셔액 등 차량 소모품 재고 관리
품목별 입고(재고 추가) / 지급(직원에게 배부) 처리
지급/입고 이력을 날짜·품목·구분별로 필터링하여 조회
CSV 내보내기로 이력 다운로드 가능
⚠️ 주의

열쇠 미반납 시 자동 알림이 발송됩니다. 출장 완료 후 반드시 반납해 주세요.

공지사항 & 알림

중요 소식과 변동 알림

공지사항
메뉴에서 📢 공지사항 선택 → 목록 확인
공지 카드 클릭 → 상세 내용 확인
이미지가 포함된 공지는 이미지 클릭 시 전체화면 확대
새 공지에는 NEW 뱃지가 표시됩니다
알림
출장/차량 탭의 🔔 알림 탭에서 확인
카테고리 필터: 열쇠, 주유, 예약, 지연, 출장예약, 출장진행, 출장완료, 소모품
모두 읽음: 현재 보이는 알림 모두 읽음 처리
30일 지난 것만 삭제: 오래된 알림 정리
전체삭제: 내 알림 전부 삭제 (다른 사람에게 영향 없음)
💡 TIP

하단 탭바의 알림 아이콘에 빨간 배지가 표시되면 미읽은 알림이 있다는 뜻입니다.

법인카드

카드 현황, 대여/반납, 이력

카드 현황
카드 현황 탭에서 전체 법인카드 상태 확인
대여중 카드: 주황색 배경으로 표시, 소지자 정보 확인 가능
보관중 카드: 초록색 강조바로 구분
카드 대여 / 반납
1카드 현황에서 원하는 카드 클릭
2보관중 카드 → 대여 버튼 → 사용 목적 입력 후 확인
3대여중 카드 → 인계(다른 직원에게) 또는 반납(재경관리부로) 선택
이력 탭
이력 탭에서 대여/반납/인계 내역 조회
카드별 필터로 특정 카드 이력만 확인 가능
CSV 내보내기 버튼으로 이력 다운로드 가능
💡 TIP

법인카드 메뉴는 카드를 소지 중이거나 카드 관리 권한이 있는 사용자에게만 표시됩니다.

설정

내 정보, 비밀번호, 로그아웃

내 정보
설정 화면 상단에서 이름, 직급, 이메일, 권한 확인
비밀번호 변경
1설정 → 비밀번호 변경 요청 클릭
2시스템관리자에게 변경 요청이 전달됩니다
3관리자 승인 후 비밀번호가 변경됩니다
사용 설명서
설정 → 사용 설명서를 눌러 앱 사용법을 다시 확인
로그아웃
설정 → 로그아웃 클릭 → 확인 모달에서 로그아웃 선택

관리자 기능 🔒 관리자

관리자 전용 기능

권한 체계 (3단계)
👤 일반 직원: 기본 기능 (재고 조회, 입출고, 출장 작성, 공지 확인)
🛡️ 관리자1: 출장/차량/재고 관리, 이력 삭제, 차량 등록
🛡️✨ 관리자2: 관리자1 + 법인카드 관리
⚡ 시스템관리자: 모든 기능 + 비밀번호 요청 처리 + 임직원 권한 변경
임직원 계정 관리
설정 → 임직원 목록에서 직원 정보 수정·삭제
역할 변경 (직원 ↔ 관리자1 ↔ 관리자2)
공지사항 작성 / 삭제
공지사항 화면 우측 상단 작성 버튼 클릭
제목, 본문(이미지 첨부 가능) 작성 후 게시
공지 카드 롱프레스 → 선택 삭제 또는 개별 삭제 가능
차량 관리
차량 탭 → 관리 탭에서 차량 등록/삭제
정비중 상태 전환, 전용 사용자 설정
보험 정보 등록 (만기 5일 전 자동 경고)
법인카드 관리 관리자2
카드 현황에서 + 버튼으로 새 카드 등록
카드 정보 수정/삭제
카드 대여/인계/반납 처리
이력 관리 (롱프레스 삭제)
재고 이력, 법인카드 이력, 소모품 이력에서 이력 카드를 길게 누르면 선택 모드 진입
개별 또는 여러 건을 선택하여 한꺼번에 삭제 가능
삭제 전 확인 모달이 표시됩니다 (실수 방지)
각 이력 탭에서 CSV 내보내기로 엑셀 다운로드 가능

홈화면에 추가하기

앱처럼 바로 실행

iPhone (Safari)
1Safari에서 이 웹앱에 접속
2하단 공유 버튼 (□↑) 클릭
3홈 화면에 추가 선택
4추가 클릭 → 홈화면에 아이콘 생성
Android (Chrome)
1Chrome에서 이 웹앱에 접속
2우측 상단 ⋮ 메뉴 → 홈 화면에 추가
3추가 클릭 → 홈화면에 아이콘 생성
💡 TIP

홈화면에서 실행하면 브라우저 주소창 없이 앱처럼 전체화면으로 사용할 수 있습니다.
