시작하기

회원가입부터 로그인까지

회원가입
1로그인 화면 하단 회원가입 클릭
2이메일, 이름, 직급, 비밀번호 입력
3보안 질문 & 답변 설정 (비밀번호 분실 시 사용)
4가입하기 클릭 → 로그인 화면으로 이동
로그인
1이메일, 비밀번호 입력
2로그인 정보 저장 체크 시 다음부터 자동 입력
3로그인 클릭
💡 TIP

비밀번호를 잊었다면 비밀번호를 잊으셨나요?를 눌러 초기화하세요.

재고 관리

부품 등록, 입출고, 이력 조회

부품 등록
1재고 목록에서 + 버튼(파란 동그라미) 클릭
2부품 유형(상용품/가공품) 선택
3종류, 품목명*, 규격, 도면번호, 메이커*, 공급처*, 단가 입력
4사진·도면 첨부 (선택)
5저장 클릭 → 재고 목록으로 이동
입출고 처리
1재고 목록에서 부품 카드 클릭 → 부품 상세 페이지
2하단 + 입고 또는 − 출고 클릭
3수량, 사유 입력 후 확인
재고 검색 & 정렬
상단 필터바 클릭 → 검색어, 정렬(종류순·이름순·수량순·최신순), 카테고리 선택
이력 탭
상단 이력 탭 → 입출고 내역 조회
입출고 / 기간 필터로 원하는 이력만 확인
💡 TIP

안전재고 이하로 떨어지면 자동 알림이 발송됩니다.

🚗

출장 / 차량

출장명령서 작성, 차량 관리

출장명령서 작성
1출장 탭 → 우측 하단 + 버튼 클릭
2목적지, 출장목적, 출발·복귀 일시 입력
3교통수단 선택 (회사 차량 / 개인 차량 / 대중교통)
4회사 차량 선택 시 → 차량 목록에서 선택 (자동 예약)
5동행자 추가 (선택)
6저장 클릭 → 상태: 예약됨
출장 상태 흐름
작성 → 예약됨 → 열쇠 수령 시 진행중 → 열쇠 반납 시 완료
차량 현황 (차량 탭)
차량별 상태(대기/예약됨/사용중/정비중) 확인
열쇠 수령: 차량 카드의 수령 버튼 클릭 → 주행거리 입력
열쇠 반납: 차량 카드의 반납 버튼 클릭 → 주행거리·주유잔량 입력
차량 상세
차량 카드 클릭 → 상세 정보 + 주유 기록 등록 + 주유 이력 확인
⚠️ 주의

열쇠 미반납 시 자동 알림이 발송됩니다. 출장 완료 후 반드시 반납해 주세요.

📢

공지사항 & 알림

중요 소식과 변동 알림

공지사항
메뉴에서 공지사항 선택 → 목록 확인
공지 카드 클릭 → 상세 내용 확인
이미지가 포함된 공지는 이미지 클릭 시 전체화면 확대
알림
출장/차량 탭의 알림 탭에서 확인
카테고리 필터: 열쇠, 주유, 예약, 지연, 출장예약, 출장진행, 출장완료
모두 읽음: 현재 보이는 알림 모두 읽음 처리
30일 지난 것만 삭제: 오래된 알림 정리
전체삭제: 내 알림 전부 삭제 (다른 사람에게 영향 없음)
💡 TIP

하단 탭바의 알림 아이콘에 빨간 배지가 표시되면 미읽은 알림이 있다는 뜻입니다.

설정

비밀번호, 임직원, 로그아웃

비밀번호 변경
1설정 → 비밀번호 변경 요청 클릭
2관리자에게 변경 요청이 전달됩니다
3관리자 승인 후 비밀번호가 변경됩니다
임직원 관리
설정 → 임직원 목록에서 동료 정보 확인
🔧

관리자 기능 🔒 관리자

관리자 전용 기능

임직원 계정 관리
설정 → 임직원 목록에서 직원 정보 수정·삭제
역할 변경 (직원 ↔ 관리자)
비밀번호 변경 요청 처리
설정 → 비밀번호 변경 요청 관리에서 승인/거절
공지사항 작성
공지사항 화면 우측 상단 ✏️ 아이콘 클릭
제목, 본문(이미지 첨부 가능) 작성 후 게시
차량 관리
차량 탭 → 관리 탭에서 차량 등록/삭제
정비중 상태 전환, 전용 사용자 설정
보험 정보 등록 (만기 5일 전 자동 경고)
이력 관리
재고 이력에서 잘못된 이력 삭제 가능
차량 이력에서 CSV 내보내기 가능

홈화면에 추가하기

앱처럼 바로 실행

iPhone (Safari)
1Safari에서 이 웹앱에 접속
2하단 공유 버튼 (□↑) 클릭
3홈 화면에 추가 선택
4추가 클릭 → 홈화면에 아이콘 생성
Android (Chrome)
1Chrome에서 이 웹앱에 접속
2우측 상단 ⋮ 메뉴 → 홈 화면에 추가
3추가 클릭 → 홈화면에 아이콘 생성
💡 TIP

홈화면에서 실행하면 브라우저 주소창 없이 앱처럼 전체화면으로 사용할 수 있습니다.
